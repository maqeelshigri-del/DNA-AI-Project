# DNA-AI-Project
A Streamlit-based DNA sequence analysis tool with machine learning models for promoter vs non-promoter and coding vs non-coding DNA classification using k-mer features and bioinformatics techniques.
# 🧬 DNA AI Project

This is a Streamlit-based bioinformatics web application that uses machine learning to classify DNA sequences.

## Features
- Promoter vs Non-promoter classification
- Coding vs Non-coding DNA prediction
- k-mer based feature extraction
- Interactive web interface

## Built With
- Python
- Streamlit
- Scikit-learn
- Bioinformatics concepts
